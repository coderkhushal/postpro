
# PostPro

## Tech Stack
 
- **Frontend Framework:** React js
- **Styling:**   tailwindCSS
- **Backend** Node.js express.js
- **Database** PostgreSQL
- **ORM** Prisma 
- **Walkthrough** - Shepherdjs 
- **Security** jsonwebtoken , bcryptjs , cryptojs 
  




## Run Locally

Clone the project

```bash
  https://github.com/coderkhushal/postpro
```

Frontend

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start React application

```bash
  npm run dev
```

Backend

```bash
  cd server
```

Install dependencies

```bash

  npm install
```

Start build typescript files

```bash
  npx tsc -b
```

Run Server

```bash
  node dist/index.js
```
